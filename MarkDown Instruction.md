## Списки

Чтобы добавить ненумерованные списки, их необходимо выделить звёздочкой (\*). Например вот так:

- Элемент 1
- Элемент 2
- Элемент 3

Чтобы добавить нумерованные списки, их необходимо выделить цифрой с точкой (1.). Например вот так:

1. Элемент 1
2. Элемент 2
3. Элемент 3
